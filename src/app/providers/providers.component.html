This page was routed to by using the routerLink directive in the header component. You can use routerLink instead of href property to navigate pages. The href property will serve all the files so the user will have to redownload them, whereas routerLink only changes the component view.

The title is displayed using interpolation. There is a field named "title" in the component that passes the value into the template, and the text is rendered wherever {{ "{{ title }\}" }} is placed.

The provider names are retrieved from a providers service. This service returns the list of provider names as an array. The providers constructor creates an instance of our providers service with dependency injection. The *ngFor directive in the template iterates over each provider name and renders a list item. The pipe formats the name into all uppercase characters.
